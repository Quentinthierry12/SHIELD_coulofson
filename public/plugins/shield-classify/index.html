Paragraph classification

Place the cursor in the paragraph to protect, then choose the level required to read it. Agents below that level will see a black bar.

Classification stamp

Inserts a colored visual stamp at the cursor (marking, not masking).

Redaction

Select the text to hide, then apply a black bar (text and background blacked out).

Watermark

Applies a diagonal watermark to every page of the document.

Signature slot

Inserts a slot where the cursor is. It will be filled with the real signature when the document is signed.

Date

Auto-filled when the document is sealed.
