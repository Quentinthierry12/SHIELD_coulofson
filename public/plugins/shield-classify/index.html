Classification du paragraphe

Place le curseur dans le paragraphe à protéger, puis choisis le niveau requis pour le lire. Les agents en dessous verront une barre noire.

Emplacement de signature

Insère un emplacement là où se trouve le curseur. Il sera rempli par la signature réelle quand le document sera signé.

Date

Se remplit automatiquement au scellement du document.
